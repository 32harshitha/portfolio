# My Portfolio Website

## Project Overview

This project is a personal portfolio website created using **HTML5** and **CSS3**.
The main purpose of this website is to present my profile, technical skills, and contact details in a professional and responsive format.

The website is designed to be simple, clean, and mobile-friendly.

---

## Objectives

* Build a personal portfolio website
* Practice HTML structure and semantic tags
* Apply CSS styling using an external stylesheet
* Create responsive design for mobile devices
* Add navigation links and hover effects

## CSS Concepts Used

### 1. CSS Selectors

Used different selectors such as:

* Element selector (`body`, `header`, `nav`)
* Class selector
* ID selector (`#about`, `#skills`, `#contact`)
* Pseudo-class (`:hover`)

### 2. CSS Variables

Used `:root` to define reusable colors.

### 3. Box Model

Used margin, padding, and border-box sizing.

### 4. Flexbox

Used for navigation layout and alignment.

### 5. Media Queries

Used to make the website responsive on smaller screens.
